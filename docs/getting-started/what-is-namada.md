# What is Namada?

**Namada** is a proof-of-stake Layer 1 blockchain focused on **multi-asset privacy**. It's designed to bring privacy to any blockchain asset through its unique Multi-Asset Shielded Pool (MASP) technology.

## Key Features

### üõ°Ô∏è Multi-Asset Shielded Pool (MASP)

Unlike traditional privacy solutions that only work with a single asset, Namada's MASP allows you to shield **any asset** from any connected blockchain. This means you can:

- Shield ETH, ATOM, OSMO, and other IBC-connected assets
- Make private transactions with any supported token
- Enjoy the same level of privacy regardless of the asset

### üîó IBC Compatible

Namada is built on the Cosmos SDK and is fully **IBC (Inter-Blockchain Communication) compatible**. This means:

- Seamless transfers between Namada and other Cosmos chains
- Access to the entire Cosmos ecosystem
- No bridges required - native IBC security

### üí∞ Shielded Rewards

Namada incentivizes privacy through **shielded rewards**:

- Earn rewards just by holding assets in the shielded pool
- The longer you shield, the more you earn
- Rewards are distributed in NAM tokens

### üèõÔ∏è Decentralized Governance

Namada features a robust governance system with:

- **Public Goods Funding (PGF)**: Community-controlled treasury for funding public goods
- **On-chain voting**: Validators and delegators can vote on proposals
- **Steward system**: Elected community members who manage PGF

## The NAM Token

**NAM** is the native token of Namada used for:

- **Staking**: Secure the network and earn rewards
- **Governance**: Vote on protocol proposals
- **Gas fees**: Pay for transactions
- **Shielded rewards**: Distributed to shielded pool participants

## Quick Facts

- **Consensus**: Proof-of-Stake (CometBFT)
- **Privacy Tech**: zk-SNARKs + MASP
- **Interoperability**: IBC + Ethereum bridge (coming)
- **Block Time**: ~6 seconds
- **Genesis**: December 2023
